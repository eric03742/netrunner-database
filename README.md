# netrunner-database

*矩阵潜袭卡牌数据仓库*

## 简介

`netrunner-database` 是用于生成《矩阵潜袭》中文卡牌数据库的工具。

本项目将数据源中提供的卡牌数据与文本数据进行处理，生成JSON格式的卡牌数据文件供其他项目使用。

## 数据源

本项目的卡牌数据来自 [NetrunnerDB](https://netrunnerdb.com/) 及其GitHub仓库 [netrunner-card-json](https://github.com/Null-Signal-Games/netrunner-cards-json)。

中文文本数据来自 [netrunner-card-text-zhCN](https://github.com/Sneakdoor-BETA/netrunner-card-text-zhCN)。
